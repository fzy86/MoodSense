All three endpoint URIs use verbs and camelCase: /uploadMood, /getMoodDistribution, /getHappyNearByLocations. REST best practice requires noun-based, lowercase, hyphen-separated resource paths (e.g., /moods, /mood-distributions, /locations/happy-nearby). The current naming makes the API look like an RPC interface rather than a resource-oriented REST API.

FAIL HTTP method semantics and status codes api_002
